- dir: ***lista arquivos***

- mkdir nome_da_pasta: ***cria pasta***

- cd nome_da_pasta: ***entra pasta***

- cd .. : ***sai pasta***

- rm nome_do_arquivo **|** ou **|** rm -r nome **|** ou **|** rm -rf nome: ***remove pasta***

- erase/del: ***deleta arquivos***

- ls: ***lista o que tiver na pasta***

- Ctrl + l ou clear: ***limpa a tela***
